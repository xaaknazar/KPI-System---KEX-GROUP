# KEX Group — KPI Калькулятор

Приложение для расчёта KPI бонусов управляющих точек KEX Group.

## Возможности

- 🧮 **Калькулятор KPI** — расчёт бонусного фонда по 7 показателям с весами 40/30/30
- 👥 **Распределение по сотрудникам** — справедливое распределение по коэффициентам ответственности
- 📊 **Шкалы KEX Group** — встроенные шкалы аудита и рейтинга
- 💾 **История расчётов** — сохранение в браузере (localStorage)
- 📄 **Экспорт в PDF и Excel** — для отчётов руководству
- ⚙️ **Настройка весов и коэффициентов** — гибкая конфигурация
- 🌙 **Тёмная тема** — фирменный стиль KEX Group

## Стек технологий

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- jsPDF + jspdf-autotable
- xlsx (SheetJS)
- lucide-react (иконки)

## Локальный запуск

```bash
npm install
npm run dev
```

## Деплой на Vercel

### Способ 1: через GitHub (рекомендуется)

1. Создайте репозиторий на GitHub и загрузите туда папку `kex-kpi`
2. Зайдите на [vercel.com](https://vercel.com) и войдите через GitHub
3. Нажмите **"Add New Project"**, выберите ваш репозиторий
4. Vercel автоматически определит Next.js — нажмите **"Deploy"**
5. Через ~1 минуту получите ссылку вида `kex-kpi.vercel.app`

### Способ 2: через CLI

```bash
npm install -g vercel
vercel login
cd kex-kpi
vercel
```

Следуйте инструкциям в терминале.

## Логика расчётов

### Структура весов KPI (40/30/30)

| Группа | Вес | KPI |
|---|---|---|
| Товарооборот | 40% | Средний чек 20% + Транзакции 20% |
| Качество | 30% | Аудит 15% + Рейтинг 15% |
| Контроль расходов | 30% | %ФОТ 10% + Текучка 10% + Списания 10% |

### Бонусный фонд точки

Формула: **Товарооборот × 3% × коэф. аудита × выполнение KPI**

### Шкала аудита (KEX Group)

| Балл | Коэф. | Действие |
|---|---|---|
| < 70% | 0 | Индивидуальное рассмотрение |
| 70-79% | 0.8 | Снижение 20% |
| 80-89% | 0.9 | Снижение 10% |
| 90-94% | 1.0 | Полный бонус |
| 95-100% | 1.1 | Поощрение +10% |

### Шкала рейтинга

| Рейтинг | Коэф. |
|---|---|
| < 4.3 | 0 |
| 4.3-4.5 | 0.5 |
| 4.5-4.7 | 0.75 |
| 4.7-5.0 | 1.0 |

### Распределение фонда между сотрудниками

```
Сумма коэф. = Σ (кол-во × коэф. позиции)
Стоимость 1 доли = Бонусный фонд ÷ Сумма коэф.
Бонус 1 человека = Коэф. позиции × Стоимость 1 доли
```
